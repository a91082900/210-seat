# 210-seat

## Usage
`seats.py [mode] [parameters]`
```
generate - generate a random seat
switch <no1> <no2> - switch seats of two students
run - apply requirements
export - export requirements to requirements.txt
print - print the current seat
debug - dry run
```

## Dependencies
- cli
  - pymongo

- web
```bash
cd 210-seats/web
npm install
```
